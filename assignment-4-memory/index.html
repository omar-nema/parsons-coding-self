What is a (single) memory?

The way I see it: a memory is a piece of data.
A single memory on its own is not significant, unless it is especially notable or traumatic.
Memory can a shape, movement, image, feeling, sound, or combination of all of these.

What is a (the faculty of) memory?

The traditional image of memory as an interconnected network mostly aligns with my thinking.
Each node represents a memory. They can be recalled in different ways.
And thus amount to different narratives.
Of course, the uniform shape of these memory nodes is reductive.
